# ➕ Add Two Numbers

A simple program that takes two numbers as input and returns their sum. This project is ideal for beginners learning basic programming concepts.

---

## 📌 Features
- Accepts two numeric inputs  
- Performs addition  
- Displays the result  
- Lightweight and easy to understand  

---

## 🛠️ Technologies Used
- Language: Python

---

## 📂 Project Structure
add-two-numbers/
│── main.py        # Main program file
│── README.md      # Project documentation

---

## 🚀 Getting Started

### 1. Clone the Repository
git clone https://github.com/your-username/add-two-numbers.git
cd add-two-numbers

### 2. Run the Program
python main.py

---

## 💻 Example Code (Python)

# Add two numbers

a = float(input("Enter first number: "))
b = float(input("Enter second number: "))

sum = a + b

print("The sum is:", sum)

---

## 🧪 Example Output
Enter first number: 5  
Enter second number: 7  
The sum is: 12
